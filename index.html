Wearday Privacy Policy

English

This app collects limited usage data to improve the app experience.

Collected data

  • App usage events (e.g. screen views, feature usage)

Data we do not collect

  • Personal information (name, email, etc.)
  • User content (items, photos, notes)

Usage

This data is anonymous and used only for analytics and product improvement.

Third parties

We use Firebase Analytics to collect usage data.

日本語

本アプリは、アプリ体験の改善を目的として限定的な利用データを収集します。

収集するデータ

  • アプリの利用状況（画面表示、機能の利用など）

収集しないデータ

  • 個人を特定できる情報（氏名、メールアドレス等）
  • ユーザーが入力した内容（アイテム名、写真、メモなど）

利用目的

これらのデータは匿名で収集され、分析およびプロダクト改善のためにのみ利用されます。

第三者提供

分析のためにFirebase Analyticsを使用しています。

한국어

이 앱은 앱 경험 개선을 위해 제한적인 사용 데이터를 수집합니다.

수집하는 데이터

  • 앱 사용 정보 (화면 조회, 기능 사용 등)

수집하지 않는 데이터

  • 개인을 식별할 수 있는 정보 (이름, 이메일 등)
  • 사용자가 입력한 내용 (아이템 이름, 사진, 메모 등)

사용 목적

이 데이터는 익명으로 수집되며 분석 및 서비스 개선에만 사용됩니다.

제3자 서비스

분석을 위해 Firebase Analytics를 사용합니다.
